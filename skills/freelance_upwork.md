# Skill: Upwork Job Application

## Policy Note
Deprecated: services are prohibited. See `/auto-dev/POLICY.md`.

## Description
Search for and apply to relevant freelance jobs on Upwork platform.

## When to Use
- Looking for quick freelance income
- Have technical skills to offer
- Need to build client base

## Prerequisites
- Upwork account (verified)
- Profile completed with skills
- Payment method set up
- Connects available (Upwork's application currency)

## Steps

### Step 1: Search for Jobs
Navigate to Upwork and search with filters:
- Category: Web Development / Data Science / Writing
- Experience: Entry/Intermediate
- Budget: $50-$500 (sweet spot for quick wins)
- Posted: Last 24 hours (less competition)

### Step 2: Evaluate Job Fit
Check these criteria before applying:
- [ ] Clear requirements (avoid vague posts)
- [ ] Client has verified payment method
- [ ] Client has hiring history
- [ ] Budget matches effort required
- [ ] Deadline is realistic

### Step 3: Craft Proposal
Structure:
1. **Hook**: Reference specific detail from job post
2. **Qualification**: Briefly state relevant experience
3. **Approach**: How you'd solve their problem
4. **Timeline**: When you can deliver
5. **Questions**: Show engagement (1-2 specific questions)

Keep proposal under 200 words. Avoid generic templates.

### Step 4: Submit and Track
- Use 2-6 connects per application
- Track applied jobs in income log
- Follow up after 48 hours if no response

## Success Criteria
- [ ] Proposal submitted successfully
- [ ] Response received within 72 hours
- [ ] Interview/contract initiated

## Common Issues
| Issue | Solution |
|-------|----------|
| No connects left | Wait for refresh or buy more |
| Profile not verified | Complete verification steps |
| Low response rate | Improve profile, target better-fit jobs |
| Competitive niche | Focus on emerging tech or niche skills |

## Income Potential
- Estimated $/hour: $25-100 depending on skill
- Difficulty: Medium
- Token cost: ~500-1000 per application cycle

## Notes
- Quality over quantity - 5 great proposals beat 20 generic ones
- Start with smaller jobs to build reviews
- Upwork takes 20% fee on first $500 with client, then decreases
- Save successful proposal templates for reuse
